# Flexbox X

```html
<row>
	<box size="2"></box>
	<box>
		<column reverse>
			<box></box>
			<box></box>
		</column>
	</box>
	<box size="3">
		<column>
			<row>
				<box></box>
				<box></box>
			</row>
			<box></box>
		</column>
	</box>
	<box size="3" align-self="flex-end"></box>
	<box></box>
</row>
```
